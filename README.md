# Kitty

A Web App written with Koa2, Webpack, Mongodb.

* Database (mongodb), ORM(mongoose)
* assets compile(webpack)
* Session with redis
* Password with bcrypt
* Testing (mocha)
* Lint (eslint)
* middleware
* server side render with react

## How to Install

```bash
 install nodejs
 install redis and mongodb
 clone this project
```
